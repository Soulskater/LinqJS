LinqJS
Functions
where
Filters on the array with the given function, that function has two parameters, current item and index.
Return value is a linq object.
Example
                        var arr=[1,2,3,4];
                        linq(arr)
                        .where(function(item, index){
                            return item % 2 === 0;
                        });
any
Iterates over the array, and returns true if any items matches with the condition
If the parameter is an object returns true if the object is in the array.
If the parameter is a function returns true if any element match against the condition.
Example
                        var arr=[1,2,3,4];
                        linq(arr)
                        .any(4);
forEach
Iterates on the array with the given function, that function has two parameters, current item and index.
This is a void function.
Example
                        var arr=[1,2,3,4];
                        linq(arr)
                        .forEach(function(item, index){
                            if(item % 2 === 0){
                                console.log("Foobar");
                            }
                        });
map
Creates a new collection, executes the given function and push the result into the new array.
Return value is a linq object.
Example
                        var arr=[1,2,3,4];
                        linq(arr)
                        .map(function(item, index){
                            return item * 2;
                        });
first
Returns with the first selected element according to the given function, otherwise throws an exception.
With no parameters it returns with the first item if the array.
Example
                        var arr=[1,2,3,4];
                        linq(arr)
                        .first(function(item, index){
                            return item % 2 === 0;
                        });
firstOrDefault
Returns with the first selected element according to the given function, otherwise null.
With no parameters it returns with the first item if the array.
Example
                        var arr=[1,2,3,4];
                        linq(arr)
                        .firstOrDefault(function(item, index){
                            return item % 2 === 0;
                        });

remove
The parameter could be a function or an item of the array.
If the parameter is a value removes it, if it's exits.
If the parameter is a function removes all of the occurencies from the array.
Example
                        var arr=[1,2,3,4];
                        linq(arr)
                        .remove(function(item){
                            return item % 2 === 0;
                        });
clear
Clear the array.
Example
                        var arr=[1,2,3,4];
                        linq(arr)
                        .clear();
